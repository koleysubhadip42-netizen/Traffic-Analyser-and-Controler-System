# Traffic-Analyser-and-Controller-System

A smart traffic management system simulation built with Python and NumPy. This project analyzes traffic density and optimizes signal timings to improve urban traffic flow.

## Features
- **Dynamic Traffic Generation:** Simulates real-time traffic density using NumPy.
- **Data Cleaning:** Handles data anomalies to ensure reliable analytics.
- **Smart Signal Timing:** Automatically calculates green light duration based on traffic thresholds.
- **Emergency Priority:** Includes logic to prioritize traffic signals for emergency vehicles.

## Technologies Used
- **Python**
- **NumPy**
